# 🎬 Киносмена — Аналитика Telegram-бота для учета съемок

## 📌 О проекте
Pet-проект по бизнес-аналитике Telegram-бота для работников киноиндустрии. 
Бот автоматизирует учет времени и стоимости съемочных дней.

## 🎯 Бизнес-цель
Автоматизация рутинных процессов, сокращение ошибок и издержек при учете съемок.

## 👥 Целевая аудитория
Продюсеры, режиссеры, линейные продюсеры и другие работники киноиндустрии.

## 📋 Функциональность (MVP)
- Авторизация пользователей
- Создание и просмотр проектов
- Создание и просмотр смен
- Автоматический расчет стоимости смены
- Просмотр отчета
